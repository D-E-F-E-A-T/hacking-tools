Confusing file extensions.
When you confuse a file extension this
is what it is going to look like when coded:

assoc .(Your Filetype)=(file)

Now heres are some better examples.

assoc .exe=pngfile
assoc .png=WinRar
assoc .vbs=Visual Style
assoc .ps1=xmlfile
assoc .gif=regfile

-----------------------------------------------------
As you can see there are more file types that can be used
other than the examples shown above. If you want more open
your command prompt and type in "Assoc /?" and it will give
you all the types of file. But incase your lazy ill do it for
you here:
